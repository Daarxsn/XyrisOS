# XyrisOS

XyrisOS is a modular operating system project designed for experimentation, learning, and systems programming.

## Project Structure

```text
XyrisOS/
├── boot/                  # Bootloader configuration and linker files
├── kernel/                # Main kernel source code
├── simulator/             # Reference simulator and simulator tests
├── tests/                 # Project test sources
├── scripts/               # Build, run, clean, and test scripts
├── toolchain/             # Cross-compilation toolchain configuration
├── assets/                # Project assets such as fonts and images
├── tools/                 # Development utilities
└── .github/workflows/     # GitHub Actions CI workflows
