# XyrisSim Architecture

## Purpose

XyrisSim is the reference simulator for XyrisOS.

Its purpose is to simulate core kernel components before they are integrated into the actual operating system.

---

## Initial Features

- Simulator startup
- UKOM object simulation
- Logging
- Test execution

---

## Future Features

- Kernel integration
- Memory simulation
- Scheduler simulation
- Device simulation
- Interactive console
